# Plan 08-01: MCP Server 分发配置

## 目标

让其他用户可以通过 Claude Desktop 使用 arcgis-agent 的 MCP Server。

## 前置条件

- Phase 5 (MCP Server) 已完成
- `arcgis_agent.mcp_server` 模块可正常导入

## 任务

### 1. 创建 MCP 配置文件示例

**文件:** `config/claude_desktop_config.example.json`

包含两种配置方式：
- 直接命令方式（需要 conda 环境）
- entry point 方式（`arcgis-agent-mcp`）

### 2. 创建环境检测脚本

**文件:** `scripts/check-env.py`

检查项：
- Python 版本 >= 3.9
- arcpy 可导入
- ArcGIS Pro 许可证有效
- 项目已安装

### 3. 创建 MCP 安装文档

**文件:** `docs/mcp-setup.md`

内容：前提条件、安装步骤、Claude Desktop 配置、验证连接、故障排除

### 4. 修正 README.md MCP 配置

当前 README 的 MCP 配置有错误（命令指向 .bat 而非正确的 entry point），需修正。

## 验证

- [ ] `scripts/check-env.py` 在正确环境下返回 0
- [ ] `scripts/check-env.py` 在缺少 arcpy 时返回非 0 并给出清晰错误
- [ ] 按照 `docs/mcp-setup.md` 步骤可在新机器上配置成功
